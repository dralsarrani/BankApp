# Bank Account System (Java)

A console-based banking application demonstrating OOP concepts including inheritance, polymorphism, and encapsulation.

## Features
- Create Savings and Checking accounts
- Deposit and withdraw money
- Add interest to savings accounts
- Overdraft protection for checking accounts
- Transaction history tracking
- Save/load accounts to file

## How to Run
```bash
javac BankingApp.java
java BankingApp
